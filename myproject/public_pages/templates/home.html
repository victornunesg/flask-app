Welcome to Flask App!

The main objective is to improve knowledge in Python through the use of libraries that help web development, as well as integration with a local database and also on Railway servers. The Blog has validations during login and operations with posts, as well as screen restrictions if the user is not logged in.

Main functionalities:

  • User creation;
  • Post creation and edition (login required);
  • User's profile edition (login required)
The development was done using mainly:
  • Python
  • Flask
  • SQLAlchemy
  • HTMl, CSS e JS/Bootstrap.
